# Weather Calendar Assitant

A Python project that gives weather-based recommendations for planned activities.

## Phase - I: Weather Recommendation Engine

This version allows the user to enter:

- an activity
- a time in 24-hour format

the program then fetches live forecast data and generates a recommendation.

## Example

```text
Enter your activity: walk
Enter a time 24-hour format (HH:MM) : 17:00

Forcasr for 17:00 in Mishawaka:
Temperature: 28.3°C
condition: Sunny
Wind: 12.6 km/h
Rain chance: 0%
Recommendation: Weather looks fine for your walk.
